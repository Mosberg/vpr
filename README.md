# Villager Professions Reimagined

This mod adds new professions and trades to Minecraft villagers, making them more diverse and interesting. Each profession has its own unique trades and items, allowing players to interact with villagers in new ways.

JSON Data-Driven: All professions and trades are defined in JSON files, making it easy for players to customize and add their own content.

## Features

- New professions such as Guard and Warrior.
- Unique trades for each profession, including weapons, armor, and rare items.
- Improved villager AI to make interactions more engaging.
